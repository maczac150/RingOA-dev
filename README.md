# FssWM
A repository for wavelet matrix using function secret sharing.

## Install

Our library has been tested on Linux. There are several library dependencies including [cryptoTools](https://github.com/ladnir/cryptoTools). 


```
git clone https://github.com/u-tmk/FssWM
cd FssWM
python build.py --setup
```

Unit tests can be run by executing the program.

```
python build.py
./out/build/<platform>/bin/test_FssWM -u
```
Other options can be seen by executing with no arguments.
